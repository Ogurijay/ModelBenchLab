# GPT WebGPU Ocean

一个基于 `WebGPU`（浏览器现代 GPU 图形接口）和 `Three.js`（三维渲染库）的实时海面模拟示例。

## 运行

```bash
npm install
npm run dev
```

## 入口

- 示例目录：`E:\Web3dTest\gpt-webgpu-ocean`
- 实现说明：`E:\Web3dTest\gpt-webgpu-ocean\docs\IMPLEMENTATION.md`
- 主代码：`E:\Web3dTest\gpt-webgpu-ocean\src\main.js`

## 浏览器要求

需要支持 WebGPU 的浏览器。推荐新版 Chrome 或 Edge，并开启硬件加速。
