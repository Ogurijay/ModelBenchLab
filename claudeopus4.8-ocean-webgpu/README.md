# 🌊 WebGPU Ocean · GPU 实时海面模拟

基于 **Three.js TSL 节点 + `WebGPURenderer`** 的实时海面模拟。
与「用 `WebGPURenderer` 渲染、却在 CPU 上逐顶点 `for` 循环算波形」的常见做法不同，
本项目把 **波形位移、法线、海水着色全部写成 TSL 节点**，由 Three.js 编译为 **WGSL** 后
真正运行在 **GPU** 上——因此能在 26 万顶点的网格上稳定跑满 60 FPS。

> 运行 `npm run dev` 后即可在浏览器中看到实时海面：低角度阳光在浪尖拉出
> 闪烁的波光带，浪尖泛起白沫，远处海面在菲涅尔反射下渐渐融入天空。

## ✨ 特性

| 能力 | 实现位置 | 说明 |
| --- | --- | --- |
| 程序化随机波谱 | 顶点着色器 `positionNode` | 8 个 Gerstner 波，**随机相位 + 方向散布 + 谱分布**（非规则正弦），带深水色散 ω=√(g·k) |
| 🎲 一键重新生成 | 重建材质 | 带种子的伪随机，确定可复现；点击即换一片全新随机海面 |
| 逐像素解析法线 | 片元着色器 `colorNode` | 用 Gerstner 导数公式一次求出法线，逐像素，水面细节锐利 |
| 菲涅尔反射 | 片元 | Schlick 近似，掠射角更亮 |
| 天空 + 太阳镜面反照 | 片元 | 沿反射方向采样同一套天空函数，物理一致的「太阳波光」 |
| 次表面散射近似 | 片元 | 迎光浪尖透出青绿 |
| 湍流海沫 | 片元 | 浪尖/陡坡决定起沫位置，再用流动噪声切成**不规则斑块**（不沿波峰成排） |
| 距离雾 | 片元 | 远处融入地平线天空色 |
| 实时参数 | uniform | 浪高 / 风速 / 陡度 / 海沫 / 日照角度，全部即时生效 |

## 🚀 运行

```bash
npm install
npm run dev      # 开发服务器
npm run build    # 生产构建到 dist/
npm run preview  # 预览构建产物
```

打开终端提示的地址（默认 `http://127.0.0.1:5173`）。

> **需要 WebGPU 浏览器**：较新版本的 Chrome / Edge，并确保「硬件加速」已开启。
> 若浏览器不支持 WebGPU，页面会显示友好的降级提示，不会白屏。

## 🎛️ 操作

- **拖拽**：旋转视角
- **滚轮**：缩放
- **右上面板**：调节浪高、风速、陡度、海沫、日照角度；暂停 / 重置视角
- **🎲 重新生成随机波浪**：换一套随机波谱，得到一片全新的海面

## 🛠️ 技术栈

- [three.js](https://threejs.org/) `0.184` — `three/webgpu` 渲染器 + `three/tsl` 着色节点
- [Vite](https://vitejs.dev/) — 构建与开发服务器
