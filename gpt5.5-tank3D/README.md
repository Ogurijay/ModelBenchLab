# GPT Tank 3D

一个经典《坦克大战》风格的 3D 网页版小游戏，包含前 10 关。

## 玩法

- `WASD` / `方向键`：移动坦克。
- `Space`：开炮。
- `P` / `Esc`：暂停。
- `R`：重开当前关卡。
- 目标：保护底部基地，击破本关全部敌方坦克。

## 规则说明

- 砖墙：可被子弹打掉，是主要可破坏掩体。
- 钢墙：不可破坏，会挡住坦克和子弹。
- 水域：坦克不能进入，子弹可以飞过。
- 树林：不会阻挡移动，但会遮挡视线。
- 冰面：坦克速度会略微提高。
- 基地：被敌方子弹击中即失败。

## 技术方案

- Three.js：3D 渲染库，负责画面、灯光、相机和模型。
- Vite：前端开发服务和构建工具。
- TypeScript：带类型检查的 JavaScript，用于减少规则和状态类 bug。
- DOM HUD：使用普通网页元素做抬头显示，避免把文本 UI 硬塞进 WebGL。

核心架构把 `simulation`（仿真/规则）和 `render`（渲染/画面）分开：游戏胜负、敌人 AI、碰撞和关卡进度都在 `src/game/simulation`，Three.js 只读取快照并同步画面。

## 本地运行

```bash
npm run dev
```

根目录门户脚本中绑定端口为 `3016`，也可以在仓库根目录运行：

```bash
npm run dev:tank3d
```

## 验证

```bash
npm run test
npm run build
```
