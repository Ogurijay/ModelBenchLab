# 任务交接文档 (HANDOVER.md) — antigravity-ocean-storm

## 1. 项目基本状态
本项目根据 `RULES.md` 规范创建，是一个基于 **Three.js (WebGL)** 实现的物理海洋与极端天气交互模拟场景。
* **开发负责人**：Antigravity (AI Coding Assistant)
* **技术核心**：
  * 海面物理：基于 **Gerstner 波 (Gerstner Waves)** 物理方程，通过顶点着色器硬件加速位移，并在 CPU 进行高度与法线同步计算提供物理浮力接口。
  * 渲染效果：利用自定义 `ShaderMaterial` 混合菲涅尔反射（Fresnel）、水下散射、泡沫贴图，呈现逼真的海洋质感。
  * 极端天气：粒子系统实现狂风暴雨（Rain）和扭转上升的巨型龙卷风（Tornado），以及程序化折线生成的雷电（Lightning）系统。
  * 交互系统：全中文的高颜值毛玻璃（Glassmorphism）UI 控制台，并可追踪小船姿态。

## 2. 详细技术方案

### A. 物理海洋 (Gerstner Waves)
为了使海浪不仅视觉真实，还能对漂浮物起物理作用，我们同时在 **GPU (Shader)** 和 **CPU (JavaScript)** 双端运行了完全一致的 Gerstner 波合成算法。
* 组合了 8 个具有不同方向、频率、振幅和相位差的简谐波。
* 解析计算切线 (Tangent) 与副切线 (Binormal) 来求出精确的法向量（Normal），使得光照计算在高耸波峰和低凹波谷处反射准确。
* 泡沫计算：在 Vertex Shader 根据 $Q_i A_i k_i$ 计算顶点的压折（Choppiness），在 Fragment Shader 提取高程和压力特征，并在波峰融合动态泡沫纹理。

### B. 极端天气系统
* **暴雨系统**：使用 `THREE.Points` 维护一个高容量的雨滴缓冲。雨滴下落方向根据当前风向进行倾斜位移，若下落至零高度（海面）则重新在云层高空生成，以降低 CPU 开销。
* **龙卷风系统**：使用多个环形多粒子带（Points）模拟龙卷风的气柱。粒子沿着阿基米德螺线旋转上升，且随着高度的增加，半径呈喇叭状向外扩张，同时加入低频正弦波使风道在空中具有蛇形扭动的动态效果。
* **闪电暴闪**：每次雷击会绘制一段基于分形中点位移算法（Midpoint Displacement）生成的折线网络，并在瞬间让场景的 `AmbientLight` 以及雾效颜色暴增，渲染惊心动魄的视觉反馈。
* **天气参数插值 (Lerp)**：支持 晴天 ➡️ 阴天 ➡️ 暴雨 ➡️ 龙卷风 的无缝渐变过渡，所有物理参数（波高、颜色、风力、雾浓度、粒子密度）均进行平滑插值。

### C. 浮力小船 (Boat Physics)
* 使用 Three.js 组合几何体动态拼接出一艘木船（含船身、桅杆和风帆）。
* 在渲染循环中，CPU 端调用 `Ocean.getBuoyancyData(x, z, time)`，获取船中心及船头、船身周围多个采样点的海面高度。
* 计算出高度平均值作为船的 $y$ 坐标，并通过采样点的相对差值估算出船的 Pitch（俯仰角）和 Roll（横滚角），使其逼真地漂浮和颠簸。
* 当切换为“龙卷风”天气时，会对小船施加一个指向龙卷风中心的引力，使其发生向心偏航。

## 3. 已知问题与后续优化
* **抗锯齿**：在极端波涛汹涌时，海浪边缘可能会有轻微走样，可以考虑开启 SMAA 或 FXAA 后期处理。
* **粒子密度限制**：为保证移动端和低配电脑的 60fps 帧率，粒子总数在代码中做了合理的上限限制。
