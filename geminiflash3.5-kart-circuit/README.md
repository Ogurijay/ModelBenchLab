# Gemini 3D 赛博朋克霓虹气垫卡丁车

本测试用例展示了 Gemini 开发的基于 **Three.js** 的 3D 未来主义气垫赛车模拟器，支持真实的漂移甩尾侧滑角物理、三维样条投影赛道、纯代码 Web Audio 合成音效和磨砂玻璃 HUD 仪表盘。

---

## 核心玩法与操控说明

在进入游戏点击 **“START”** 解锁音频并起步后，可以使用以下按键进行驾驶：

* **【W】/【↑】**：加速推进
* **【S】/【↓】**：减速 / 倒车
* **【A】/【D】/【←】/【→】**：向左 / 向右转向
* **【Space】(空格键)**：**甩尾漂移蓄力**
* **【R】**：复位重置（当卡丁车卡住或冲出界时）

### 🌟 核心技巧：漂移多段小喷 (Mini-Turbo)
在高速过弯（速度 > 7 KM/H）且持续按着转向键时，**长按空格键**即可让赛车进入甩尾漂移状态。
* **侧滑姿态**：赛车船体会自动呈现朝向弯内斜侧偏的甩尾角度，且两侧尾喷管下方会渲染出动态的发光**霓虹双轨胎痕**。
* **蓄力升级**：只要处于漂移状态，时速表顶部的能量条就会不断积攒。
  * **1段小喷 (蓝色)**：蓄力槽满 38%，电荷变为蓝色。
  * **2段小喷 (紫色)**：蓄力槽满 85%，电荷变为紫色，车身尾部喷射紫色电火花。
* **释放喷气**：在蓄力变为蓝色或紫色后，**松开空格键**即可瞬间爆发出对应级别的 Boost 喷射加速，此时摄像机视野（FOV）会自动拉伸，产生强烈的极速纵深感。

## 技术架构与物理方程

### 1. 3D 样条赛道投影器 (`src/simulation/trackSpline.js`)
我们使用 3D Catmull-Rom 闭合曲线绘制赛道轴线。
* 通过每帧对赛车位置进行数值投影搜索（粗筛 + 局部二分细化），求解出最近的样条参数 $t \in [0, 1]$、切线正方向和距离中心线的垂直偏差。
* **出界与逆行判定**：如果偏差大于 `ROAD_HALF_WIDTH = 9.5`，物理引擎判定为出界（越野），滚动阻力剧增，极速被死锁在低频；通过赛车 heading 与样条切线的夹角余弦，可高精度判定赛车是否处于“逆行（Wrong Way）”状态，触发发红的警告霓虹线条。

### 2. 反重力气垫悬浮与侧偏角 (`src/simulation/kart.js`)
* 赛车 Mesh 在 Y 轴上基于正弦函数做高频微弱波动，模拟反重力浮空感。
* 漂移时引入侧偏角 $SlipAngle = -Steer \times 0.38$，使实际质心运动方向比船头朝向偏外侧，逼真表现出横向漂移的惯性离心滑动。

### 3. 纯代码 Web Audio 声音合成 (`src/audio.js`)
不依赖任何外部音频文件，完全利用浏览器音频电路：
* **引擎咆哮**：Sawtooth（锯齿波，提供粗糙质感）与 Triangle（三角波，亚八度深沉低音）双振荡器，音调与音量与当前油门和车速线性变频。
* **小喷爆发**：通过高通滤波器对白噪声进行滤波，结合指数衰减的 Gain，完美还原出氮气爆破和排气嘶嘶声。

## 运行与验证

在 `geminiflash3.5-kart-circuit/` 目录下打开终端：

### 1. 运行物理与大门状态单元测试
```bash
npm run test
```
* 验证二阶积分阻力、漂移侧偏角计算、多级小喷时速暴增、样条投影和进圈计数正确性。

### 2. 启动本地开发服务器
```bash
npm run dev
```
* 访问本地地址即可体验游戏。
* **加速道具带 (Boost Pad)**：赛道上设计了两个粉红发光格栅加速带（分别位于 $t \approx 0.39$ 和 $t \approx 0.73$），踩上后会获得强力加速。
